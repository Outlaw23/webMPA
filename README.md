# 🎵 Jukebox Playlist 🎵

---

### Brief explanation

In this project, I created a full-stack Jukebox Playlist application where users can browse songs, filter them by genre, and create their own playlists.

The goal of this project is to build a modern web application using a Spring Boot backend and a React frontend while applying software development principles such as separation of concerns, database migrations, and ORM-based data management.

During the development of this project, I gained experience working with Java, Spring Boot, React, and MariaDB/MySQL. The application follows a layered architecture and uses a relational database to manage songs, genres, users, and playlists.

This project was developed as part of a school assignment and demonstrates the use of REST APIs, database relationships, and modern frontend development.
